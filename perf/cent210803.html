OpenDDS - Performance Testing Environments

OCI "Centipede" Virtual Test Environment 2021

The test environment includes the test hosts and the network equipment connecting the hosts. All hosts in the tests were the same and a single network was used to connect them.
  1. Test Hosts
    • 3 Dell R820 machines
    • 4 Intel Xeon E5-4640v2 (2.2GHz) CPUs in each R820
    • 20mb cache per CPU
    • 110 GB RAM per R280
    • Dell X530 network cards
    • Each R820 is running the ProxMox Virtual Environment (PVE)
    • Each PVE machine is configured to run 10 Virtual machines, each running CentOS Linux release 7.9.2009 (Core) with 4 CPUs and 2GB RAM
  2. Test Network
    • Each PVE Machine Is Running Linux Bridge Devices for the VMs, connected to a Ubiquiti UniFI US-16-XG 16-port managed switch (10G)
